# Roadmap

## Phase 1 — Bootstrap
Set up Next.js project, lint/format/test baselines, and Playwright + axe-core harness.

## Phase 2 — Inputs & Scans
Implement dual URL/upload input flow, run paired scans with timeouts, and persist snapshots for demo cases.

## Phase 3 — Diff & Report
Build violation diffing logic, severity grouping, and UI that highlights new regressions only with WCAG references.

## Phase 4 — Guidance & Export
Add fix suggestion templates, component-hinting, and exportable markdown/PR comment output.

## Phase 5 — Polishing
Seed sample datasets, add loading/error states, accessibility of the tool itself, and document architecture/demo usage.
